How It Works

This demo shows how an ECU senses input from a temperature sensor (like a coolant temperature sensor).

  • Pull-Up Circuit: Voltage decreases as temperature increases (resistance decreases)
  • Pull-Down Circuit: Voltage increases as temperature increases (resistance decreases)
  • When the sensor is disconnected, pull-up circuits read 5V and pull-down circuits read 0V
  • The ECU uses a lookup table to convert voltage readings to temperature values

Adjust the temperature slider to see how it affects the sensor resistance and voltage reading.
